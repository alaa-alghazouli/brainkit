# Roadmap

## Current (v2.0)

- One production skill: `threejs-performance-optimizer`
- CLI and JavaScript API for listing and installing skills
- Packaged `.skill` artifact format
- Zero-dependency, ESM-only package for Node.js 18+

## Next (Planned)

- Additional skills using the existing multi-skill architecture
- Tool registry population in `tools/manifest.json`

## Exploring

- Community-contributed skills
- Skill versioning and update checking
- Cross-platform skill formats

Ideas welcome through discussions and issues.
